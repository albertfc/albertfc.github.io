WordleCAT Helper

This calculator will help you figure out Wordle (also Absurdle, Hello Wordl, or Seven Wordles) puzzles. The word list contains all the possible words that can be used to solve Wordle puzzles. The list can be filtered by entering information from Wordle about which letters are correct and incorrect.

Green Letters

Type correct letters that are colored green by Wordle into the green boxes. For example, after typing "T" into the first green box, the word list gets filtered to only show words that start with "T."

Yellow Letters

Type misplaced letters that are colored yellow by Wordle into the yellow boxes. For example, after typing "A" into the second yellow box, the word list gets filtered to only show words that contain an "A" but not as their second letter. More than one letter can be entered into each yellow box because you may make additional guesses in Wordle that give you another yellow letter in the same position. For example, after typing "AS" into the second yollow box, the word list get filtered to only show words that contain both "A" and "S," but neither as the second letter.

Dark Gray letters

Type wrong letters that are colored dark gray by Wordle into the dark gray box. The position of wrong letters within the word doesn't matter, so all wrong letters should be entered into a single input here. For example, after typing "ABC" into the dark gray box, the word list gets filtered to only show words that don't contain "A," "B," or "C".

Word List Order

The word list is ordered by how helpful guessing the word is likely to be. Words at the beginning of the list use letters that are lots of the remaining words. Words at the end of the list have duplicate letters or use letters that are in few remaining words.

Starting Over

Use the "Clear" button under the word list to start over with no letters entered.
